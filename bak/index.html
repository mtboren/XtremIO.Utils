Xtremio.Utils

A PowerShell module for interacting with EMC XtremIO storage arrays via their XtremIO Management Server REST API

View on GitHub Download .zip Download .tar.gz

XtremIO.Utils PowerShell module

Some examples and their sample output (see each cmdlet's help for more examples):
